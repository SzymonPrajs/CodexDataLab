## App Controls (Planned / Reserved)

These are intended to be implemented for automation (not wired up yet):

- Ctrl+G: Full plot
- Ctrl+T: Full chat
- Ctrl+B: Split view
- Ctrl+F: Toggle menu
- Tab: Toggle focus (menu/chat)
- Arrow Up/Down: Navigate plot list
- Enter: Pin plot / send chat
