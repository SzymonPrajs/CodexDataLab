# Contributing

Thanks for your interest in contributing to CodexDataLab!

By participating in this project, you agree to follow the Code of Conduct in
`CODE_OF_CONDUCT.md`.

## Development setup

- Create and activate a virtual environment.
- Install in editable mode:
  - `pip install -e .`

## Running locally

- Run the app:
  - `python -m codexdatalab`

## Submitting changes

- Keep changes focused and well-described.
- Add/adjust tests when applicable.
- Open a pull request with a clear description of what and why.
